# Development Principles

## Philosophy

Learning Journal Engine is designed as an engine,
not a single workbook.

---

## Principle 1
Content First

Content should exist independently of any renderer.

---

## Principle 2
Engine Independent

Modules must not depend on the engine.

---

## Principle 3
Documentation First

Every major feature starts with documentation.

---

## Principle 4
One Source of Truth

A learning module should have only one source.

---

## Principle 5
Export Everywhere

The same content should be exportable to:

- PDF
- GoodNotes
- Notion
- Web

without modification.
